此插件可以部分内容加密；写文章或页面时，只需加短代码标签[pw key="密码"]需要加密的内容[/pw]

== Description ==
大家都知道，其实wordpress自带就有文章和页面加密功能

但是，WordPress自带的文章或页面加密码，只能是整编文章或页面，此插件可以做到加密某一段落的内容，这样可以有效，防止网络爬虫、采集、蜘蛛等。

或者由于现在网络净网行动不便于直接公布相关链接、视频、图片。需要“部分内容加密”来隐藏

== Installation ==
只需加短代码标签[pw key="密码"]需要加密的内容[/pw]

== Frequently Asked Questions ==
1. 如何设置密码
在短代码pw后面key的值，就是你要设置的密码

2. 如何告知访客密码 
1，打电话、电邮给博主
2，问博主朋友
3，QQ群、TG群组
4，公众号“XX”回复，验证码
等等多种方式，主要是配合私域流量转换
